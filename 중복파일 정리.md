# rclone 사용
- 설치
```
curl https://rclone.org/install.sh | sudo bash
rclone --version
```

- 개념
  - 소스와 타겟 폴더에 대해 같은이름 파일 확인(동일 파일이라도 이름이 같으면 skip)
  - 이름이 같으면 파일 크기 비교
  - 수정시간으로 파일 검사 --checksum을 붙이면 MD5/SHA-1 해시 비교
  - --dry-run 옵션 수행시 미수행(시뮬레이션 모드)
  - --log-file=결과물.txt -v 옵션 주면 로그 기록
  - rclone move는 이동, copy는 복사(사본 유지)

```
sudo rclone copy "백업" "원본" --ignore-existing --suffix _copy --suffix-keep-extension --ignore-times --checksum --log-file="rclone_report.txt" -v
```
